## Set Up

### Locally

1. Clone the repo
   ```sh
   git@github.com:thomedpete/Stretch-Tech.git
   ```
2. Enter the directory and install NPM packages
   ```sh
   npm install
   npm start
   ``` 
3. Enter the following url in your browser: http://localhost:3000/
4. Explore the website
